About

Recipes

This website contains my family's favorite recipes, with a focus on Dairy-Free and Soy-Free recipes.

With different food allergies in our household, we focus on using allergy-friendly ingredients for our recipes.

Technology

This website is built using Flask (web framework written in Python).

Flask is typically used for developing dynamic website. For this website, Frozen-Flask is used to create the static site.

The static site is then hosted on Netlify.


To learn more about how this website is created and deployed, check out the blog post on TestDriven.io:

Generating a Static Site with Flask and Deploying it to Netlify

All of the source code for this website can be found on GitLab:

GitLab - Flask Recipe App
